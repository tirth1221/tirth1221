## 🚀 Featured Projects

### 🔹 DIGIPIN API Tool
A Node.js + Google Maps-based geocode encoder/decoder for 10-character alphanumeric geolocation codes.  
➡️ Integrates Maps, Autocomplete, Reverse Geocoding, and Directions between DIGIPINs.

### 🔹 ESP32 Wi-Fi Control Panel
An ESP32-based local server that lets users control devices (like LEDs, buzzers, servos) via a website hosted on the chip itself.  
➡️ No external Wi-Fi needed!

### 🔹 RFID Smart Access System
A smart access control project using **ESP32 + MFRC522 RFID**, ideal for schools or hotel room automation.
